About This Lab

Java Maven web application (JSP + Tomcat + MySQL) for Lab_12_Z.

Calculator page: addition table (base 7). Home form logs number, square, and parity to MySQL.

Run the Java application

Build and run with Docker locally:

mvn clean package -DskipTests
docker compose up --build

Then open: http://localhost:8081/WebApp/
